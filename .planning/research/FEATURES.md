# Feature Research: QSB Ask

## Table Stakes For This Product

### Authentication And Roles

- Email/password sign-in for organisers, moderators, and speakers.
- Password reset.
- Event-level roles: Organiser, Moderator, Speaker.
- Participant access without account creation.

### Events

- Event creation.
- Join code/link.
- Event settings.
- Close/archive event.

### Moderated Q&A

- Pending, Live, Answered, Archived states.
- Approve, dismiss, edit, archive, restore, mark answered.
- Edit history and moderation action history.
- Upvotes.
- Search and sort.
- Live update indicators.

### Presenter And Presentation

- Presenter View for approved questions only.
- Presentation View for survey result charts only.
- Large readable text.
- No moderation/admin controls in display views.

### Surveys

- Multiple choice.
- Multiple select.
- Rating.
- Open text.
- Draft, published, closed states.
- Result visibility toggle, hidden from participants by default.
- Results dashboard with chart and data views.

### Export

- CSV exports for questions, moderation history, and survey responses.

## Explicit Deferrals

- Excel export.
- Question labels.
- Downvotes.
- Audience-visible highlighted question.
- Administrator role across all events.
- Offline moderation action queue.
- AI-assisted survey creation.
- Word cloud, quiz, idea board, and gamification.
